Tasks Completed

Task 1: Iris Flower Classification (Machine Learning)

Objective:
To build a machine learning model that classifies iris flowers into three species (Setosa, Versicolor, Virginica) based on their physical measurements.

Key Steps:

* Loaded and explored the Iris dataset
* Performed data preprocessing
* Trained classification models
* Evaluated model performance
* Made predictions on new input data

Algorithms Used:
Logistic Regression / KNN / Decision Tree (as applicable)



 Task 2: Unemployment Analysis in India (Data Science Project)

Objective:
To analyze unemployment trends in India, especially during the COVID-19 period, using real-world data.

Key Steps:

* Exploratory Data Analysis (EDA)
